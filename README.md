# Clinica Frontend

This is the frontend application for the Clinic Management System, built with **React**, **Vite**, and **TypeScript**. It communicates with the backend via REST API.

## technologies Used

### Core
- **React** - UI Library (v19)
- **Vite** - Build tool and development server
- **TypeScript** - Static typing

### UI & Styling
- **Material UI (MUI)** - Component library (v7)
- **@mui/x-date-pickers** - Advanced date picking components
- **Emotion** - CSS-in-JS styling engine
- **Lucide React** / **MUI Icons** - Icons

### State & Data Fetching
- **Axios** - HTTP client
- **Better Auth** - Authentication client
- **React Router DOM** - Client-side routing

### specialized Components
- **FullCalendar** - Interactive calendar for appointments
- **Recharts** - Charting library for dashboards
- ** @dnd-kit** & **@hello-pangea/dnd** - Drag and drop functionality

### Utilities
- **Date-fns** - Date manipulation
- **Jwt-decode** - JWT token parsing

## Getting Started

1. Install dependencies:
   ```bash
   bun install
   ```

2. Run development server:
   ```bash
   bun dev
   ```
